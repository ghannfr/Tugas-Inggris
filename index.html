"UKRI LEARNING" application is designed to support lecture activities at Universitas Kebangsaan Republik Indonesia (UKRI) effectively and efficiently. This application is designed with a simple and good design that makes users comfortable in accessing. The "UKRI LEARNING" application has advantages for students and lecturers, because by using this application lecturers can create an attendance list so that students can attend online and lecturers can check it online too.

Not only that, the "UKRI LEARNING" application can also create assignments, give assignments, collect assignments, and assess assignments. No less interesting, this application has been designed to be able to communicate between lecturers and students or students with other students using the message feature. With the "UKRI LEARNING" application, it can make it easier for students and lecturers to be able to access lecture activities easily. Our Jargon: Learn to serve, achieve knowledge for the nation!.
